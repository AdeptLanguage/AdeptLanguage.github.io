In Development https://adeptlanguage.github.io/v1/adept.stash

FAQ

Where is 'adept.config'?

You can find which folder 'adept.config' lives in by running 'adept --root'

What's a stash?

A stash is a network of packages that can be downloaded via 'adept install <package_name>'
